# Northcoders News API

To begin, create 2 .env files:

.env.test
.env.development

In .env.test, write:

PGDATABASE=nc_news_test

In .env.development, write:

PGDATABASE=nc_news_test

Double check that these .env files are .gitignored.

Run npm install
